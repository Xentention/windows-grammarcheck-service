## 1. О программе

Сервис исправляет текст на русском языке в два этапа:

1. **`ai-forever/sage-fredt5-large`** — модель типа «из последовательности в
   последовательность», обученная специально для исправления орфографии русского
   языка. Здесь исправляется большинство ошибок. Модель квантована в ONNX
   (INT8 или INT4) для быстрой работы на CPU.
2. **`RUPunct/RUPunct_big`** + детерминированная постобработка — дополнительно
   расставляет пунктуацию, регистр и пробелы, покрывая случаи, упущенные на
   первом этапе.

Текст обрабатывается по абзацам и строкам. Все выполняется локально; сервис
поднимается автоматически как служба Windows.

Поддерживаемые варианты:

- **Веб-интерфейс** с крупным дизайном, опцией высокого контраста и поддерживающий скрин-ридеры.
- **Горячие клавиши** — возможность привязать комбинации `Ctrl + Win + Shift + Alt + C` и `Ctrl + Win + Shift + Alt + Z` к действиям исправить текст/отменить исправление.
- **Скрипты** (`.vbs`) для собственной автоматизации и привязки к сторонним программам
  (например, [Tobii Communicator 5](#4-привязка-к-tobii-communicator-5)).

Установщик содержит модель внутри себя — качать что-либо ещё не нужно. Доступны
две сборки, отличающиеся квантованием модели орфографии:

| Сборка                   | Ссылка                                                                                                                                                 | Когда выбирать                                                      |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------- |
| **INT8** (рекомендуется) | [**RuGrammarCheck-Setup-int8.exe**](https://github.com/Xentention/windows-grammarcheck-service/releases/download/v0.0.1/RuGrammarCheck-Setup-int8.exe) | Выше качество исправлений. Значение по умолчанию.                   |
| **INT4**                 | [**RuGrammarCheck-Setup-int4.exe**](https://github.com/Xentention/windows-grammarcheck-service/releases/download/v0.0.1/RuGrammarCheck-Setup-int4.exe) | Меньше размер и требования к памяти, чуть быстрее — ценой качества. |

Ссылки выше всегда ведут на **последний релиз**. Все версии и примечания к
выпускам — на странице
[Releases](https://github.com/Xentention/windows-grammarcheck-service/releases).

> Установщик весит порядка 1.5 ГБ, так как включает квантованную модель. Требуются
> права администратора и 64-разрядная Windows 10/11.

## 3. Установка и работа

### Установка

Запустите скачанный `RuGrammarCheck-Setup-*.exe`:

- создает папку по адресу `C:\ProgramData\RuGrammarCheck`;
- регистрирует службу Windows `RuGrammarCheck` (автостарт при загрузке системы);
- по желанию настраивает **горячие клавиши**;
- создает ярлык в меню «Пуск» и на рабочем столе.

После установки модели несколько секунд загружается в память, после чего
программа готова к работе.

### Веб-интерфейс (UI)

Откройте **RuGrammarCheck**

1. Вставьте или наберите текст в поле **«Ввод»**.
2. Нажмите **«Исправить»**.
3. Скопируйте результат из поля **«Вывод»**

### Горячие клавиши

Если горячие клавиши включены при установке

| Действие                                      | Сочетание                        |
| --------------------------------------------- | -------------------------------- |
| Исправить текст в буфере обмена               | **Ctrl + Win + Shift + Alt + C** |
| Откатить исправление (вернуть исходный текст) | **Ctrl + Win + Shift + Alt + Z** |

Порядок работы:

1. выделите и скопируйте текст (Ctrl+C);
2. нажмите **Ctrl + Win + Shift + Alt + C**;
3. немного подождите;
4. вставьте исправленный текст (Ctrl+V).

### Скрипты

| Скрипт          | Путь                                                            | Что делает                                                   |
| --------------- | --------------------------------------------------------------- | ------------------------------------------------------------ |
| Исправить буфер | `C:\ProgramData\RuGrammarCheck\clipboard\correct-clipboard.vbs` | Исправляет текущее содержимое буфера обмена.                 |
| Откатить        | `C:\ProgramData\RuGrammarCheck\clipboard\revert-clipboard.vbs`  | Возвращает в буфер исходный текст до последнего исправления. |

Исходный текст для отката сохраняется в `%LOCALAPPDATA%\grammarcheck\last_input.txt`.

### Управление сервисом

Служба называется `RuGrammarCheck`. Управлять можно через системное приложение «Службы» или:

```
"C:\ProgramData\RuGrammarCheck\nssm.exe" status  RuGrammarCheck
"C:\ProgramData\RuGrammarCheck\nssm.exe" restart RuGrammarCheck
```

Логи (с ротацией) находятся в `C:\ProgramData\RuGrammarCheck\logs\RuGrammarCheck\`.
Удаление происходит стандартным для приложений Windows способом.

## 4. Привязка к Tobii Communicator 5

> ℹ️ Данный раздел еще не протестирован

**Пути к скриптам**:

- Исправить: `C:\ProgramData\RuGrammarCheck\clipboard\correct-clipboard.vbs`
- Отменить: `C:\ProgramData\RuGrammarCheck\clipboard\revert-clipboard.vbs`

**Как привязать.** В Communicator 5 перейдите в режим редактирования страницы,
Если действие требует именно исполняемый файл (`.exe`), запускайте скрипт через
`wscript.exe`:

`C:\Windows\System32\wscript.exe "C:\ProgramData\RuGrammarCheck\clipboard\correct-clipboard.vbs"`

**Последовательность для кнопки «Исправить»:**

1. Скопировать текст области сообщения в буфер обмена;
2. Запустить `correct-clipboard.vbs`.
3. Вставить результат обратно (Ctrl+V).

> ⚠️ Исправление **не мгновенное**. Не вставляйте текст в том же действии сразу
> после запуска скрипта — вставится ещё неисправленный текст.

Для отмены исправления аналогичные действия, но скрипт `C:\ProgramData\RuGrammarCheck\clipboard\revert-clipboard.vbs`. Действие мгновенное.

## 5. Технические детали

### Последовательность обработки

1. Входной текст разбивается на абзацы;
2. Каждый абзац отправляется в модель орфографии
   `sage-fredt5-large` (жадное декодирование, `num_beams=1`). На строку —
   тайм-аут 30 с; при превышении или ошибке возвращается исходная строка.
3. Абзац проходит через `RUPunct_big` (token-classification) и
   детерминированную постобработку: расстановка регистра и пунктуации,
   нормализация пробелов вокруг скобок и кавычек, схлопывание дублирующихся
   знаков.
4. Абзацы собираются обратно с сохранением исходных переносов строк.

### Модели

| Роль       | Модель                                             | Формат            | Размер на диске                 |
| ---------- | -------------------------------------------------- | ----------------- | ------------------------------- |
| Орфография | `ai-forever/sage-fredt5-large`                     | ONNX, INT8 / INT4 | ~1.3 ГБ (int8) / ~1.2 ГБ (int4) |
| Пунктуация | `RUPunct/RUPunct_big` (BERT, token classification) | ONNX              | ~177 МБ                         |

Загрузка моделей — через `optimum.onnxruntime` (`ORTModelForSeq2SeqLM` и
`ORTModelForTokenClassification`) на `CPUExecutionProvider`. Обе модели грузятся
параллельно при старте.

### HTTP API

**Flask + Waitress**, слушает `127.0.0.1:<PORT>`.

| Метод и путь    | Назначение                                                                                                   |
| --------------- | ------------------------------------------------------------------------------------------------------------ |
| `GET /`         | UI                                                                                                           |
| `POST /correct` | in: `{"text": "..."}` out: `{"corrected": "...", "elapsed_seconds": 1.23, "elapsed_seconds_str": "1.230s"}`. |
| `GET /health`   | Healthcheck: `{"status": "ok"}`.                                                                             |

KeepAliveWarmer каждые 240 секунды простоя прогоняет фиктивный запрос,
чтобы держать ONNX-ядра теплыми и не замедлять первый реальный запрос.
