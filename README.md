# Корректор орфографии и пунктуации на базе sage-fredt5-large + RUPunct_big)

Локальный сервис для исправления орфографии и пунктуации русского текста
на базе моделей `ai-forever/sage-fredt5-large` и `RUPunct/RUPunct_big`, 
квантованных в ONNX INT8 для оптимизации работы на CPU.
`ai-forever/sage-fredt5-large` также доступна в ONNX INT4.

Текст обрабатывается по абзацам, после обработки моделью, каждый абзац затем проходит через
`RUPunct/RUPunct_big` и детерминированную постобработку для дополнительной расстановки пунктуации.

## Запуск

Пока доступен в dev, запускается на `http://localhost:$PORT/` (по умолчанию
http://localhost:8501/).

```
docker compose up -d --build web
```
